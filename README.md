## 🚀 Cách Sử Dụng

1. **Đọc theo thứ tự**: Các bài học được sắp xếp từ cơ bản đến nâng cao
2. **Thực hành ngay**: Mỗi bài có code mẫu - hãy chạy thử trên máy
3. **Làm bài tập**: Cuối mỗi bài có bài tập thực hành, cố gắng tự làm trước khi xem gợi ý
4. **Tham khảo links**: Mỗi bài có links tài liệu chính thức để đọc thêm

## 🛠 Yêu Cầu Cơ Bản

- Node.js >= 18 (LTS)
- npm hoặc yarn
- Expo CLI: `npm install -g expo-cli`
- Expo Go app trên điện thoại
- VS Code + Extensions: ESLint, Prettier, React Native Tools

## 📌 Lưu Ý

- Tất cả code mẫu đều viết bằng **TypeScript**
- Sử dụng **React Native** với **Expo**
- Mỗi bài học độc lập, có thể đọc riêng lẻ

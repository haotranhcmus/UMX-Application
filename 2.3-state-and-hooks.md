# 2.3 State & Hooks

> **Mục tiêu**: Quản lý state trong component, sử dụng thành thạo các hooks cơ bản.

---

## 📖 Tổng Quan

**State** là dữ liệu "có thể thay đổi" bên trong component. Khi state thay đổi, component sẽ **re-render** (vẽ lại UI). **Hooks** là các hàm đặc biệt của React cho phép bạn sử dụng state và các tính năng khác trong function components.

**Quy tắc Hooks:**

1. ☝️ Chỉ gọi hooks **ở top level** — KHÔNG gọi trong if, for, hay nested functions
2. ⚛️ Chỉ gọi hooks **trong React function components** hoặc custom hooks

## 1. useState

### 1.1 Khởi tạo State

```tsx
import React, { useState } from "react";
import { View, Text, TouchableOpacity, StyleSheet } from "react-native";

const BasicStateDemo = () => {
  // Cú pháp: const [state, setState] = useState(initialValue);
  //          [giá trị, hàm cập nhật] = useState(giá trị ban đầu)
  const [count, setCount] = useState(0); // number
  const [name, setName] = useState("Hào"); // string
  const [isVisible, setIsVisible] = useState(true); // boolean
  const [items, setItems] = useState<string[]>([]); // array (TypeScript)

  return (
    <View style={styles.container}>
      <Text>Count: {count}</Text>
      <Text>Name: {name}</Text>
      <Text>Visible: {isVisible ? "Yes" : "No"}</Text>
      <Text>Items: {items.length}</Text>
    </View>
  );
};

const styles = StyleSheet.create({
  container: { padding: 16, gap: 8 },
});
```

### 1.3 Update State với Function (Functional Update)

**Quan trọng**: Khi state mới phụ thuộc vào state cũ, dùng **function form**.

```tsx
import React, { useState } from "react";
import { View, Text, TouchableOpacity, StyleSheet } from "react-native";

const FunctionalUpdateDemo = () => {
  const [count, setCount] = useState(0);

  // ❌ SAI — có thể bị "stale state" khi click nhanh
  const incrementBad = () => {
    setCount(count + 1); // Dùng count hiện tại (có thể cũ)
  };

  // ✅ ĐÚNG — luôn dùng giá trị mới nhất
  const incrementGood = () => {
    setCount((prev) => prev + 1); // prev luôn là giá trị mới nhất
  };

  // ✅ Increment 3 lần liên tiếp
  const incrementThree = () => {
    // Dùng functional update — mỗi lần đều dùng prev mới nhất
    setCount((prev) => prev + 1); // prev = 0 → 1
    setCount((prev) => prev + 1); // prev = 1 → 2
    setCount((prev) => prev + 1); // prev = 2 → 3
  };

  // ❌ SAI — cả 3 dùng cùng 1 giá trị count cũ
  const incrementThreeBad = () => {
    setCount(count + 1); // count = 0 → set 1
    setCount(count + 1); // count vẫn = 0 → set 1
    setCount(count + 1); // count vẫn = 0 → set 1
    // Kết quả: chỉ tăng 1!
  };

  return (
    <View style={styles.container}>
      <Text style={styles.value}>Count: {count}</Text>
      <TouchableOpacity style={styles.btn} onPress={incrementGood}>
        <Text style={styles.btnText}>+1 (Correct)</Text>
      </TouchableOpacity>
      <TouchableOpacity style={styles.btn} onPress={incrementThree}>
        <Text style={styles.btnText}>+3 (Correct)</Text>
      </TouchableOpacity>
      <TouchableOpacity style={styles.btn} onPress={() => setCount(0)}>
        <Text style={styles.btnText}>Reset</Text>
      </TouchableOpacity>
    </View>
  );
};

const styles = StyleSheet.create({
  container: { padding: 16, gap: 8 },
  value: { fontSize: 28, fontWeight: "bold", textAlign: "center" },
  btn: { backgroundColor: "#007AFF", padding: 12, borderRadius: 8 },
  btnText: { color: "#fff", textAlign: "center", fontWeight: "600" },
});
```

### 1.4 State với Object/Array (Immutable Update)

**Quy tắc vàng**: KHÔNG mutate state trực tiếp. Luôn tạo **bản sao mới**.

```tsx
import React, { useState } from "react";
import {
  View,
  Text,
  TouchableOpacity,
  TextInput,
  StyleSheet,
} from "react-native";

// ========== Object State ==========
interface UserProfile {
  name: string;
  email: string;
  age: number;
  address: {
    city: string;
    district: string;
  };
}

const ObjectStateDemo = () => {
  const [user, setUser] = useState<UserProfile>({
    name: "Hào",
    email: "hao@example.com",
    age: 22,
    address: { city: "HCM", district: "Q1" },
  });

  // ✅ Update 1 field — spread operator
  const updateName = (newName: string) => {
    setUser((prev) => ({ ...prev, name: newName }));
  };

  // ✅ Update nested object — spread từng level
  const updateCity = (newCity: string) => {
    setUser((prev) => ({
      ...prev,
      address: { ...prev.address, city: newCity },
    }));
  };

  // ❌ SAI — mutate trực tiếp
  // user.name = 'Minh';           // ❌ Không re-render!
  // user.address.city = 'HN';     // ❌ Không re-render!

  return (
    <View style={styles.container}>
      <Text>Name: {user.name}</Text>
      <Text>Email: {user.email}</Text>
      <Text>City: {user.address.city}</Text>

      <TouchableOpacity style={styles.btn} onPress={() => updateName("Minh")}>
        <Text style={styles.btnText}>Change name to Minh</Text>
      </TouchableOpacity>
      <TouchableOpacity style={styles.btn} onPress={() => updateCity("Hà Nội")}>
        <Text style={styles.btnText}>Change city to Hà Nội</Text>
      </TouchableOpacity>
    </View>
  );
};

// ========== Array State ==========
const ArrayStateDemo = () => {
  const [todos, setTodos] = useState<string[]>(["Học React", "Đọc docs"]);

  // ✅ Thêm item
  const addTodo = (text: string) => {
    setTodos((prev) => [...prev, text]);
  };

  // ✅ Xóa item theo index
  const removeTodo = (index: number) => {
    setTodos((prev) => prev.filter((_, i) => i !== index));
  };

  // ✅ Update item theo index
  const updateTodo = (index: number, newText: string) => {
    setTodos((prev) => prev.map((item, i) => (i === index ? newText : item)));
  };

  // ❌ SAI — mutate trực tiếp
  // todos.push('New todo');           // ❌
  // todos[0] = 'Updated';            // ❌
  // todos.splice(0, 1);              // ❌

  return (
    <View style={styles.container}>
      {todos.map((todo, index) => (
        <View key={index} style={styles.todoRow}>
          <Text style={{ flex: 1 }}>{todo}</Text>
          <TouchableOpacity onPress={() => removeTodo(index)}>
            <Text>🗑</Text>
          </TouchableOpacity>
        </View>
      ))}
      <TouchableOpacity style={styles.btn} onPress={() => addTodo("New Todo")}>
        <Text style={styles.btnText}>Add Todo</Text>
      </TouchableOpacity>
    </View>
  );
};

const styles = StyleSheet.create({
  container: { padding: 16, gap: 8 },
  btn: { backgroundColor: "#007AFF", padding: 12, borderRadius: 8 },
  btnText: { color: "#fff", textAlign: "center", fontWeight: "600" },
  todoRow: {
    flexDirection: "row",
    alignItems: "center",
    padding: 8,
    backgroundColor: "#f9f9f9",
    borderRadius: 6,
  },
});
```

### 1.5 Lazy Initial State

Khi initial value cần tính toán nặng, dùng **function initializer** — chỉ chạy 1 lần.

```tsx
import React, { useState } from "react";

// ❌ Chạy mỗi lần render (dù chỉ dùng kết quả lần đầu)
const BadExample = () => {
  const [data, setData] = useState(expensiveComputation());
  // ...
};

// ✅ Chỉ chạy 1 lần khi mount
const GoodExample = () => {
  const [data, setData] = useState(() => expensiveComputation());
  // ...
};

const expensiveComputation = () => {
  console.log("Heavy computation running...");
  // Giả sử tính toán phức tạp
  return Array.from({ length: 1000 }, (_, i) => ({
    id: i,
    value: Math.random(),
  }));
};
```

## 2. useEffect

### 2.1 Basic Effect

`useEffect` cho phép thực hiện **side effects** — fetch data, subscriptions, DOM manipulation, timers.

```tsx
import React, { useState, useEffect } from "react";
import { View, Text, StyleSheet } from "react-native";

const EffectDemo = () => {
  const [count, setCount] = useState(0);

  // ========== Pattern 1: Chạy SAU MỖI render ==========
  useEffect(() => {
    console.log("Component rendered! Count:", count);
  });
  // ⚠️ Cẩn thận: chạy liên tục — ít khi dùng

  // ========== Pattern 2: Chạy 1 LẦN khi mount ==========
  useEffect(() => {
    console.log("Component mounted! (chỉ 1 lần)");
    // Fetch initial data, setup subscriptions...
  }, []);
  // 👆 [] = empty dependency array = chỉ chạy khi mount

  // ========== Pattern 3: Chạy khi dependency thay đổi ==========
  useEffect(() => {
    console.log("Count changed to:", count);
    // Fetch new data based on count, update title...
  }, [count]);
  // 👆 [count] = chạy khi count thay đổi

  return (
    <View style={styles.container}>
      <Text>Count: {count}</Text>
    </View>
  );
};

const styles = StyleSheet.create({
  container: { padding: 16 },
});
```

### 2.3 Cleanup Function

**Cleanup** chạy khi:

1. Component **unmount** (bị xóa khỏi screen)
2. **Trước khi** effect chạy lại (dependency thay đổi)

```tsx
import React, { useState, useEffect } from "react";
import { View, Text, StyleSheet, AppState, AppStateStatus } from "react-native";

const CleanupDemo = () => {
  const [seconds, setSeconds] = useState(0);
  const [appState, setAppState] = useState(AppState.currentState);

  // ========== Timer — cleanup khi unmount ==========
  useEffect(() => {
    console.log("Starting timer...");
    const interval = setInterval(() => {
      setSeconds((prev) => prev + 1);
    }, 1000);

    // 🧹 Cleanup: clear interval khi unmount
    return () => {
      console.log("Cleaning up timer...");
      clearInterval(interval);
    };
  }, []);

  // ========== Event listener — cleanup khi unmount ==========
  useEffect(() => {
    const subscription = AppState.addEventListener(
      "change",
      (nextState: AppStateStatus) => {
        setAppState(nextState);
        console.log("App state:", nextState);
      },
    );

    // 🧹 Cleanup: remove listener
    return () => {
      subscription.remove();
    };
  }, []);

  return (
    <View style={styles.container}>
      <Text>⏱ Timer: {seconds}s</Text>
      <Text>📱 App State: {appState}</Text>
    </View>
  );
};

const styles = StyleSheet.create({
  container: { padding: 16, gap: 8 },
});
```

### 2.4 Common Patterns: Fetch Data

```tsx
import React, { useState, useEffect } from "react";
import {
  View,
  Text,
  FlatList,
  ActivityIndicator,
  StyleSheet,
} from "react-native";

interface Post {
  id: number;
  title: string;
  body: string;
}

const FetchDataDemo = () => {
  const [posts, setPosts] = useState<Post[]>([]);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState<string | null>(null);

  useEffect(() => {
    let isMounted = true; // 👈 Flag để tránh update state sau unmount

    const fetchPosts = async () => {
      try {
        setLoading(true);
        setError(null);

        const response = await fetch(
          "https://jsonplaceholder.typicode.com/posts?_limit=10",
        );

        if (!response.ok) {
          throw new Error(`HTTP error! status: ${response.status}`);
        }

        const data: Post[] = await response.json();

        // ✅ Chỉ update state nếu component vẫn mounted
        if (isMounted) {
          setPosts(data);
        }
      } catch (err) {
        if (isMounted) {
          setError(err instanceof Error ? err.message : "Unknown error");
        }
      } finally {
        if (isMounted) {
          setLoading(false);
        }
      }
    };

    fetchPosts();

    // 🧹 Cleanup: set flag khi unmount
    return () => {
      isMounted = false;
    };
  }, []);

  if (loading) {
    return (
      <View style={styles.center}>
        <ActivityIndicator size="large" color="#007AFF" />
        <Text>Loading posts...</Text>
      </View>
    );
  }

  if (error) {
    return (
      <View style={styles.center}>
        <Text style={styles.error}>Error: {error}</Text>
      </View>
    );
  }

  return (
    <FlatList
      data={posts}
      keyExtractor={(item) => item.id.toString()}
      renderItem={({ item }) => (
        <View style={styles.postCard}>
          <Text style={styles.postTitle}>{item.title}</Text>
          <Text style={styles.postBody} numberOfLines={2}>
            {item.body}
          </Text>
        </View>
      )}
      contentContainerStyle={styles.list}
    />
  );
};

const styles = StyleSheet.create({
  center: { flex: 1, justifyContent: "center", alignItems: "center", gap: 8 },
  error: { color: "red", fontSize: 16 },
  list: { padding: 16 },
  postCard: {
    backgroundColor: "#fff",
    padding: 12,
    marginBottom: 8,
    borderRadius: 8,
    borderLeftWidth: 4,
    borderLeftColor: "#007AFF",
  },
  postTitle: { fontSize: 16, fontWeight: "600" },
  postBody: { color: "#666", marginTop: 4 },
});

export default FetchDataDemo;
```

## 3. useRef

### 3.1 Ref cho Mutable Value (không trigger re-render)

```tsx
import React, { useState, useRef, useEffect } from "react";
import { View, Text, TouchableOpacity, StyleSheet } from "react-native";

const UseRefDemo = () => {
  const [count, setCount] = useState(0);

  // useRef lưu giá trị KHÔNG gây re-render khi thay đổi
  const renderCount = useRef(0);
  const previousCount = useRef(0);
  const intervalRef = useRef<ReturnType<typeof setInterval> | null>(null);

  // Đếm số lần render
  useEffect(() => {
    renderCount.current += 1;
  });

  // Lưu giá trị count trước đó
  useEffect(() => {
    previousCount.current = count;
  }, [count]);

  // Timer với ref
  const startTimer = () => {
    intervalRef.current = setInterval(() => {
      setCount((prev) => prev + 1);
    }, 1000);
  };

  const stopTimer = () => {
    if (intervalRef.current) {
      clearInterval(intervalRef.current);
      intervalRef.current = null;
    }
  };

  return (
    <View style={styles.container}>
      <Text>Count: {count}</Text>
      <Text>Previous: {previousCount.current}</Text>
      <Text>Renders: {renderCount.current}</Text>

      <View style={styles.row}>
        <TouchableOpacity style={styles.btn} onPress={startTimer}>
          <Text style={styles.btnText}>▶ Start</Text>
        </TouchableOpacity>
        <TouchableOpacity style={styles.btn} onPress={stopTimer}>
          <Text style={styles.btnText}>⏹ Stop</Text>
        </TouchableOpacity>
      </View>
    </View>
  );
};

const styles = StyleSheet.create({
  container: { padding: 16, gap: 8 },
  row: { flexDirection: "row", gap: 8 },
  btn: { backgroundColor: "#007AFF", padding: 12, borderRadius: 8 },
  btnText: { color: "#fff", fontWeight: "600" },
});
```

## 4. useCallback

Memoize function để tránh tạo lại mỗi render — hữu ích khi truyền function làm prop cho child component.

```tsx
import React, { useState, useCallback } from "react";
import {
  View,
  Text,
  TouchableOpacity,
  FlatList,
  StyleSheet,
} from "react-native";

// Child component dùng React.memo
const TodoItem = React.memo(
  ({ text, onDelete }: { text: string; onDelete: () => void }) => {
    console.log(`Rendering: ${text}`); // Log để thấy re-render
    return (
      <View style={styles.todoItem}>
        <Text style={{ flex: 1 }}>{text}</Text>
        <TouchableOpacity onPress={onDelete}>
          <Text>🗑</Text>
        </TouchableOpacity>
      </View>
    );
  },
);

const UseCallbackDemo = () => {
  const [todos, setTodos] = useState(["Learn React", "Learn RN", "Build App"]);
  const [count, setCount] = useState(0);

  // ❌ Không có useCallback — tạo hàm mới mỗi render
  // → TodoItem sẽ re-render dù data không đổi
  // const handleDelete = (index: number) => {
  //   setTodos(prev => prev.filter((_, i) => i !== index));
  // };

  // ✅ Có useCallback — hàm chỉ tạo lại khi dependencies thay đổi
  const handleDelete = useCallback((index: number) => {
    setTodos((prev) => prev.filter((_, i) => i !== index));
  }, []); // [] vì dùng functional update, không phụ thuộc biến ngoài

  return (
    <View style={styles.container}>
      {/* count thay đổi → parent re-render → nhưng TodoItem không re-render */}
      <Text>Count: {count}</Text>
      <TouchableOpacity
        style={styles.btn}
        onPress={() => setCount((c) => c + 1)}
      >
        <Text style={styles.btnText}>Increment (won't re-render todos)</Text>
      </TouchableOpacity>

      {todos.map((todo, index) => (
        <TodoItem key={todo} text={todo} onDelete={() => handleDelete(index)} />
      ))}
    </View>
  );
};

const styles = StyleSheet.create({
  container: { padding: 16, gap: 8 },
  btn: { backgroundColor: "#007AFF", padding: 12, borderRadius: 8 },
  btnText: { color: "#fff", textAlign: "center", fontWeight: "600" },
  todoItem: {
    flexDirection: "row",
    alignItems: "center",
    padding: 12,
    backgroundColor: "#f9f9f9",
    borderRadius: 6,
  },
});
```

**Khi nào cần `useCallback`?**

- ✅ Truyền callback cho child component dùng `React.memo`
- ✅ Callback là dependency của `useEffect`
- ❌ Không cần cho mọi function — chỉ khi có vấn đề performance

## 5. useMemo

Memoize **giá trị tính toán** — chỉ tính lại khi dependencies thay đổi.

```tsx
import React, { useState, useMemo } from "react";
import { View, Text, TextInput, FlatList, StyleSheet } from "react-native";

interface Product {
  id: string;
  name: string;
  price: number;
  category: string;
}

const UseMemoDemo = () => {
  const [searchTerm, setSearchTerm] = useState("");
  const [sortBy, setSortBy] = useState<"name" | "price">("name");
  const [count, setCount] = useState(0);

  // Giả sử danh sách sản phẩm rất lớn
  const products: Product[] = [
    { id: "1", name: "iPhone", price: 25000000, category: "Phone" },
    { id: "2", name: "MacBook", price: 55000000, category: "Laptop" },
    { id: "3", name: "iPad", price: 18000000, category: "Tablet" },
    { id: "4", name: "AirPods", price: 6000000, category: "Audio" },
    { id: "5", name: "Apple Watch", price: 12000000, category: "Watch" },
  ];

  // ✅ useMemo — chỉ tính lại khi searchTerm hoặc sortBy thay đổi
  const filteredAndSorted = useMemo(() => {
    console.log("Computing filtered & sorted list...");

    const filtered = products.filter((p) =>
      p.name.toLowerCase().includes(searchTerm.toLowerCase()),
    );

    return filtered.sort((a, b) => {
      if (sortBy === "name") return a.name.localeCompare(b.name);
      return a.price - b.price;
    });
  }, [searchTerm, sortBy]);
  // 👆 KHÔNG chạy lại khi count thay đổi

  // ✅ useMemo cho giá trị tổng hợp
  const totalPrice = useMemo(() => {
    return filteredAndSorted.reduce((sum, p) => sum + p.price, 0);
  }, [filteredAndSorted]);

  return (
    <View style={styles.container}>
      <TextInput
        style={styles.input}
        value={searchTerm}
        onChangeText={setSearchTerm}
        placeholder="Tìm sản phẩm..."
      />

      <Text>Tổng: {totalPrice.toLocaleString("vi-VN")}đ</Text>
      <Text>Count (unrelated): {count}</Text>

      {filteredAndSorted.map((item) => (
        <View key={item.id} style={styles.product}>
          <Text style={styles.productName}>{item.name}</Text>
          <Text style={styles.productPrice}>
            {item.price.toLocaleString("vi-VN")}đ
          </Text>
        </View>
      ))}
    </View>
  );
};

const styles = StyleSheet.create({
  container: { padding: 16, gap: 8 },
  input: { borderWidth: 1, borderColor: "#ddd", borderRadius: 8, padding: 10 },
  product: {
    flexDirection: "row",
    justifyContent: "space-between",
    padding: 12,
    backgroundColor: "#f9f9f9",
    borderRadius: 6,
  },
  productName: { fontSize: 16, fontWeight: "500" },
  productPrice: { color: "#007AFF", fontWeight: "600" },
});
```

### Bài 2: Debounced Search (useRef)

```tsx
// exercises/DebouncedSearch.tsx
import React, { useState, useRef, useEffect } from "react";
import {
  View,
  Text,
  TextInput,
  FlatList,
  ActivityIndicator,
  StyleSheet,
} from "react-native";

const DebouncedSearch = () => {
  const [query, setQuery] = useState("");
  const [results, setResults] = useState<string[]>([]);
  const [loading, setLoading] = useState(false);
  const timerRef = useRef<ReturnType<typeof setTimeout> | null>(null);

  // Debounce search — đợi 500ms sau khi user ngừng gõ
  useEffect(() => {
    // Clear timer trước đó
    if (timerRef.current) {
      clearTimeout(timerRef.current);
    }

    if (!query.trim()) {
      setResults([]);
      return;
    }

    // Set timer mới
    timerRef.current = setTimeout(async () => {
      setLoading(true);
      try {
        // Giả lập API call
        await new Promise((resolve) => setTimeout(resolve, 500));
        const mockResults = [
          `${query} - result 1`,
          `${query} - result 2`,
          `${query} - result 3`,
        ];
        setResults(mockResults);
      } finally {
        setLoading(false);
      }
    }, 500);

    // Cleanup timer khi unmount hoặc query thay đổi
    return () => {
      if (timerRef.current) {
        clearTimeout(timerRef.current);
      }
    };
  }, [query]);

  return (
    <View style={styles.container}>
      <TextInput
        style={styles.input}
        value={query}
        onChangeText={setQuery}
        placeholder="Search with debounce..."
      />

      {loading && <ActivityIndicator style={styles.loader} />}

      <FlatList
        data={results}
        keyExtractor={(item, index) => index.toString()}
        renderItem={({ item }) => <Text style={styles.result}>{item}</Text>}
        ListEmptyComponent={
          !loading && query ? (
            <Text style={styles.empty}>No results</Text>
          ) : null
        }
      />
    </View>
  );
};

const styles = StyleSheet.create({
  container: { padding: 16 },
  input: {
    borderWidth: 1,
    borderColor: "#ddd",
    borderRadius: 8,
    padding: 12,
    fontSize: 16,
  },
  loader: { marginTop: 12 },
  result: { padding: 12, borderBottomWidth: 1, borderBottomColor: "#eee" },
  empty: { textAlign: "center", color: "#999", marginTop: 16 },
});

export default DebouncedSearch;
```

## ✅ Checklist Tự Đánh Giá

### useState

- [ ] Khởi tạo state: `const [state, setState] = useState(initialValue)`
- [ ] Update state với value mới
- [ ] Update state với function: `setState(prev => prev + 1)`
- [ ] State với object/array (immutable update)
- [ ] Lazy initial state: `useState(() => expensiveComputation())`

### useEffect

- [ ] Basic effect: `useEffect(() => { ... })`
- [ ] Dependency array
- [ ] Empty dependency `[]` — run once on mount
- [ ] Cleanup function: `return () => { cleanup }`
- [ ] Multiple useEffects
- [ ] Common patterns: fetch data, event listeners

### useRef

- [ ] Tạo ref: `const ref = useRef(initialValue)`
- [ ] Ref cho component (TextInput, ScrollView...)
- [ ] Ref cho mutable value (không trigger re-render)

### useCallback & useMemo

- [ ] `useCallback` — memoize function
- [ ] `useMemo` — memoize computed value
- [ ] Hiểu khi nào cần dùng

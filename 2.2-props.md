# 2.2 Props

> **Mục tiêu**: Truyền data giữa components, validate props với TypeScript.

---

## 📖 Tổng Quan

**Props** (Properties) là cách truyền dữ liệu từ component cha xuống component con. Props hoạt động giống tham số (parameters) của hàm — bạn gọi component và truyền dữ liệu vào.

**Đặc điểm quan trọng:**

- 📤 **One-way data flow** — Dữ liệu chỉ chảy từ cha → con
- 🔒 **Readonly** — Component con KHÔNG ĐƯỢC sửa props
- 🏷️ **Type-safe** — Dùng TypeScript để validate props

---

## 1. Passing Props

### 1.1 Truyền Props Cơ Bản

```tsx
import React from "react";
import { View, Text, StyleSheet } from "react-native";

// Truyền props giống HTML attributes
const App = () => {
  return (
    <View style={styles.container}>
      {/* String prop */}
      <Greeting name="Hào" />
      {/* Number prop — dùng {} */}
      <Greeting name="Minh" age={25} />
      {/* Boolean prop */}
      <Greeting name="Lan" age={23} isVIP={true} />
      <Greeting name="Tú" isVIP /> {/* Shorthand = true */}
      {/* Object prop */}
      <Greeting name="An" address={{ city: "HCM", district: "Q1" }} />
      {/* Array prop */}
      <Greeting name="Bình" hobbies={["Code", "Game"]} />
      {/* Function prop */}
      <Greeting name="Chi" onGreet={() => console.log("Hello!")} />
    </View>
  );
};

const styles = StyleSheet.create({
  container: { padding: 16, gap: 8 },
});
```

## 4. Props là Readonly (One-Way Data Flow)

```tsx
import React from "react";
import { View, Text, TouchableOpacity } from "react-native";

interface CounterDisplayProps {
  count: number;
}

// ❌ SAI — KHÔNG BAO GIỜ sửa props
const BadCounter = ({ count }: CounterDisplayProps) => {
  // count = count + 1;  // ❌ TypeError: Cannot assign to read only property
  // props.count++;       // ❌ Tương tự

  return <Text>Count: {count}</Text>;
};

// ✅ ĐÚNG — Nếu cần thay đổi, dùng callback props
interface CounterProps {
  count: number;
  onIncrement: () => void; // Callback để thông báo cha
  onDecrement: () => void;
}

const Counter = ({ count, onIncrement, onDecrement }: CounterProps) => {
  return (
    <View style={{ flexDirection: "row", alignItems: "center", gap: 12 }}>
      <TouchableOpacity onPress={onDecrement}>
        <Text style={{ fontSize: 24 }}>➖</Text>
      </TouchableOpacity>
      <Text style={{ fontSize: 20, fontWeight: "bold" }}>{count}</Text>
      <TouchableOpacity onPress={onIncrement}>
        <Text style={{ fontSize: 24 }}>➕</Text>
      </TouchableOpacity>
    </View>
  );
};
```

**Tại sao readonly?**

- Đảm bảo **dữ liệu chảy 1 chiều** (predictable)
- Component cha **kiểm soát** dữ liệu
- Dễ **debug** — biết data đến từ đâu

## 5. Passing Functions as Props

Truyền hàm callback từ cha xuống con — cho phép con "giao tiếp ngược" với cha.

```tsx
import React, { useState } from "react";
import {
  View,
  Text,
  StyleSheet,
  TouchableOpacity,
  TextInput,
} from "react-native";

// ========== CHILD COMPONENT ==========
interface TodoItemProps {
  id: string;
  text: string;
  completed: boolean;
  onToggle: (id: string) => void; // Callback: toggle complete
  onDelete: (id: string) => void; // Callback: delete item
  onEdit: (id: string, newText: string) => void; // Callback: edit text
}

const TodoItem = ({
  id,
  text,
  completed,
  onToggle,
  onDelete,
  onEdit,
}: TodoItemProps) => {
  return (
    <View style={styles.todoItem}>
      {/* Gọi callback với data */}
      <TouchableOpacity onPress={() => onToggle(id)}>
        <Text style={{ fontSize: 20 }}>{completed ? "✅" : "⬜"}</Text>
      </TouchableOpacity>

      <Text style={[styles.todoText, completed && styles.completedText]}>
        {text}
      </Text>

      <TouchableOpacity onPress={() => onDelete(id)}>
        <Text style={{ fontSize: 16 }}>🗑</Text>
      </TouchableOpacity>
    </View>
  );
};

// ========== PARENT COMPONENT ==========
interface Todo {
  id: string;
  text: string;
  completed: boolean;
}

const TodoApp = () => {
  const [todos, setTodos] = useState<Todo[]>([
    { id: "1", text: "Học React Native", completed: false },
    { id: "2", text: "Đọc tài liệu Props", completed: true },
    { id: "3", text: "Làm bài tập", completed: false },
  ]);

  // Các hàm xử lý — truyền xuống child
  const handleToggle = (id: string) => {
    setTodos((prev) =>
      prev.map((todo) =>
        todo.id === id ? { ...todo, completed: !todo.completed } : todo,
      ),
    );
  };

  const handleDelete = (id: string) => {
    setTodos((prev) => prev.filter((todo) => todo.id !== id));
  };

  const handleEdit = (id: string, newText: string) => {
    setTodos((prev) =>
      prev.map((todo) => (todo.id === id ? { ...todo, text: newText } : todo)),
    );
  };

  return (
    <View style={styles.container}>
      <Text style={styles.title}>📋 Todo List</Text>
      {todos.map((todo) => (
        <TodoItem
          key={todo.id}
          id={todo.id}
          text={todo.text}
          completed={todo.completed}
          onToggle={handleToggle} // ← Truyền function
          onDelete={handleDelete} // ← Truyền function
          onEdit={handleEdit} // ← Truyền function
        />
      ))}
    </View>
  );
};

const styles = StyleSheet.create({
  container: { padding: 16 },
  title: { fontSize: 24, fontWeight: "bold", marginBottom: 16 },
  todoItem: {
    flexDirection: "row",
    alignItems: "center",
    padding: 12,
    backgroundColor: "#fff",
    marginBottom: 8,
    borderRadius: 8,
    gap: 12,
  },
  todoText: { flex: 1, fontSize: 16 },
  completedText: { textDecorationLine: "line-through", color: "#999" },
});

export default TodoApp;
```

## 6. Spread Props

Dùng spread operator `{...}` để truyền tất cả props cùng lúc.

```tsx
import React from "react";
import {
  View,
  Text,
  StyleSheet,
  TouchableOpacity,
  TouchableOpacityProps,
} from "react-native";

// ========== Spread Props cơ bản ==========
interface UserProps {
  name: string;
  email: string;
  role: string;
}

const UserCard = (props: UserProps) => (
  <View>
    <Text>{props.name}</Text>
    <Text>{props.email}</Text>
  </View>
);

const App = () => {
  const user: UserProps = {
    name: "Hào",
    email: "hao@example.com",
    role: "Developer",
  };

  // ✅ Spread — truyền tất cả properties
  return <UserCard {...user} />;

  // Tương đương với:
  // return <UserCard name={user.name} email={user.email} role={user.role} />;
};

// ========== Spread Props nâng cao: Wrapper Component ==========

// Custom Button wraps TouchableOpacity
interface CustomButtonProps extends TouchableOpacityProps {
  title: string;
  variant?: "primary" | "secondary" | "danger";
}

const CustomButton = ({
  title,
  variant = "primary",
  style,
  ...rest
}: CustomButtonProps) => {
  const variantStyle = {
    primary: { backgroundColor: "#007AFF" },
    secondary: { backgroundColor: "#6c757d" },
    danger: { backgroundColor: "#dc3545" },
  };

  return (
    // ...rest spread tất cả props còn lại của TouchableOpacity
    <TouchableOpacity
      style={[styles.button, variantStyle[variant], style]}
      {...rest}
    >
      <Text style={styles.buttonText}>{title}</Text>
    </TouchableOpacity>
  );
};

// Sử dụng — onPress, disabled, etc. được spread qua
const SpreadDemo = () => (
  <View style={styles.container}>
    <CustomButton
      title="Submit"
      variant="primary"
      onPress={() => console.log("Pressed!")}
    />
    <CustomButton
      title="Delete"
      variant="danger"
      disabled={true}
      onPress={() => console.log("Delete!")}
    />
  </View>
);

const styles = StyleSheet.create({
  container: { padding: 16, gap: 12 },
  button: { padding: 14, borderRadius: 8, alignItems: "center" },
  buttonText: { color: "#fff", fontWeight: "600", fontSize: 16 },
});

export default SpreadDemo;
```

### 7.2 Type vs Interface

```tsx
// Interface — có thể extend, merge
interface ButtonProps {
  title: string;
  onPress: () => void;
}

interface IconButtonProps extends ButtonProps {
  icon: string; // Thêm prop
}

// Type — linh hoạt hơn với union, intersection
type Variant = "primary" | "secondary" | "danger";
type Size = "sm" | "md" | "lg";

type StyledButtonProps = ButtonProps & {
  variant: Variant;
  size: Size;
};

// 💡 Quy tắc: Dùng interface cho props, type cho union/utility types
```

```tsx
  disabled?: boolean;
  onPress: () => void;
}

const Button = ({
  title,
  variant = "primary",
  size = "medium",
  disabled = false,
  onPress,

    primary: { bg: "#007AFF", text: "#fff", border: "#007AFF" },
```

### Bài 3: Callback Props

**Yêu cầu**: Parent truyền callback, child gọi callback với data.

```tsx
// exercises/CallbackProps.tsx
import React, { useState } from "react";
import {
  View,
  Text,
  StyleSheet,
  TextInput,
  TouchableOpacity,
} from "react-native";

// ===== CHILD: Search Input =====
interface SearchInputProps {
  placeholder?: string;
  onSearch: (query: string) => void; // Callback trả về search query
}

const SearchInput = ({
  placeholder = "Tìm kiếm...",
  onSearch,
}: SearchInputProps) => {
  const [text, setText] = useState("");

  const handleSubmit = () => {
    if (text.trim()) {
      onSearch(text.trim()); // Gọi callback với data
      setText("");
    }
  };

  return (
    <View style={styles.searchRow}>
      <TextInput
        style={styles.input}
        value={text}
        onChangeText={setText}
        placeholder={placeholder}
        onSubmitEditing={handleSubmit}
      />
      <TouchableOpacity style={styles.searchBtn} onPress={handleSubmit}>
        <Text style={styles.searchBtnText}>🔍</Text>
      </TouchableOpacity>
    </View>
  );
};

// ===== PARENT =====
const CallbackPropsExercise = () => {
  const [searches, setSearches] = useState<string[]>([]);

  // Parent xử lý data từ child

};
```
